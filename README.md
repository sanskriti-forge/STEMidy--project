# STEMidy 🚀

A fun, space-themed gamified learning platform where students become astronauts and explore the Solar System while studying.

### What is STEMidy?
STEMidy turns learning into an adventure. You pick your class, choose a subject, and travel through the planets by completing quizzes. Score perfectly to unlock the next planet!

It's designed to make studying more engaging and enjoyable with a beautiful cosmic theme.

### Features

- **Solar System Progression** – Unlock planets (Mercury to Saturn) by scoring 100% in quizzes
- **8 Subjects**: Physics, Math, Chemistry, English, Geography, Biology, Social Studies, General Knowledge
- **Interactive Quizzes** with Previous/Next navigation
- **Dashboard** showing your learning progress, study hours, and weekly timetable
- **Pomodoro Timer** for focused study sessions
- **Quick Notes** section
- **Cosmic Tutor** – A chatbot to help with doubts
- **Student Profile** with overall progress

### How to Run

1. Download or clone this project
2. Open the `frontend/index.html` file directly in your browser
3. No installation needed — it works offline

### How to Play

- Start from the *Dashboard*
- Go to *Solar System*
- Click on any unlocked planet
- Answer the quiz questions
- Get 100% to unlock the next planet in the solar system

### Tech Stack

- HTML5
- CSS3 (with space-themed animations)
- Vanilla JavaScript
